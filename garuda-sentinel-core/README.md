# GARUDA Frontend

This folder contains the main React showcase experience for GARUDA.

## What is inside

- cinematic landing page
- cyber-themed visual effects
- product storytelling sections
- threat dashboard components
- polished Tailwind and shadcn/ui interface

## Local development

```bash
npm install
npm run dev
```

Default local URL: `http://localhost:8080`

## Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
